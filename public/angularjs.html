Configuración y Publicación de un Sitio Usando AngularJS

  1. Instalación de Node.js y npm

    Descarga e instala Node.js desde su sitio web oficial.

  2. Instalación de Angular CLI

    Abre una terminal y ejecuta el siguiente comando para instalar Angular CLI:

    npm install -g @angular/cli

  3. Creación de un Nuevo Proyecto Angular

    Crea un nuevo proyecto Angular:

    ng new mi-proyecto-angular

    Sigue las instrucciones para configurar el proyecto.

  4. Desarrollo y Prueba de la Aplicación

    Navega al directorio del proyecto y ejecuta el servidor de desarrollo:

    cd mi-proyecto-angular
    ng serve

    Abre un navegador web y navega a http://localhost:4200 para ver tu aplicación en funcionamiento.

  5. Construcción de la Aplicación para Producción

    Construye la aplicación para producción:

    ng build --prod

    Los archivos de producción se encontrarán en el directorio dist/.

  6. Despliegue del Sitio

    Para desplegar tu sitio, puedes usar servicios como Firebase, Netlify, o GitHub Pages. Aquí tienes una guía rápida para Firebase:

    1. Instala la CLI de Firebase:
      2. npm install -g firebase-tools
    2. Inicia sesión en Firebase:
      3. firebase login
    3. Inicializa tu proyecto Firebase en el directorio del proyecto Angular:
      4. firebase init
    4. Selecciona "Hosting" y sigue las instrucciones para configurar el despliegue.
    5. Despliega tu aplicación:
      6. firebase deploy
