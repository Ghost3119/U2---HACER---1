Configuración y Publicación de un Sitio Usando Node.js

  1. Instalación de Node.js

    Descarga e instala Node.js desde su sitio web oficial.

  2. Configuración del Proyecto

    Abre una terminal y crea una carpeta para tu proyecto:

    mkdir mi-proyecto
cd mi-proyecto

    Inicializa el proyecto con npm:

    npm init -y

  3. Instalación de Dependencias

    Instala Express.js, un framework web para Node.js:

    npm install express

  4. Creación del Servidor

    Crea un archivo index.js con el siguiente contenido:

    const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('¡Hola Mundo!');
});

app.listen(port, () => {
  console.log(`Servidor escuchando en http://localhost:${port}`);
});

  5. Ejecutar el Servidor

    En la terminal, ejecuta el servidor:

    node index.js

    Abre un navegador web y navega a http://localhost:3000 para ver tu sitio en funcionamiento.

  6. Despliegue del Sitio

    Para desplegar tu sitio, puedes usar servicios como Heroku, AWS, o DigitalOcean. Aquí tienes una guía rápida para Heroku:

    1. Instala el CLI de Heroku.
    2. Inicia sesión en Heroku desde la terminal:
      3. heroku login
    3. Crea una nueva aplicación en Heroku:
      4. heroku create
    4. Sube tu proyecto a Heroku:
      5. git init
git add .
git commit -m "Initial commit"
git push heroku master
    5. Abre tu aplicación desplegada:
      6. heroku open
