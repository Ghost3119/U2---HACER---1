Configuración y Publicación de un Sitio Usando Laravel

  1. Instalación de Composer

    Descarga e instala Composer desde su sitio web oficial.

  2. Creación de un Nuevo Proyecto Laravel

    Abre una terminal y ejecuta el siguiente comando para crear un nuevo proyecto Laravel:

    composer create-project --prefer-dist laravel/laravel mi-proyecto-laravel

  3. Configuración del Servidor de Desarrollo

    Navega al directorio del proyecto y ejecuta el servidor de desarrollo:

    cd mi-proyecto-laravel
php artisan serve

    Abre un navegador web y navega a http://localhost:8000 para ver tu aplicación en funcionamiento.

  4. Configuración del Entorno

    Edita el archivo .env en el directorio del proyecto para configurar la base de datos y otras variables de entorno según tus necesidades.

  5. Despliegue del Sitio

    Para desplegar tu sitio, puedes usar servicios como Heroku, AWS, o un servidor VPS. Aquí tienes una guía rápida para Heroku:

    1. Instala el CLI de Heroku.
    2. Inicia sesión en Heroku desde la terminal:
      3. heroku login
    3. Crea un archivo Procfile en el directorio del proyecto con el siguiente contenido: 
      4. web: vendor/bin/heroku-php-apache2 public/
    4. Inicializa un repositorio git en el directorio del proyecto:
      5. git init
git add .
git commit -m "Initial commit"
    5. Crea una nueva aplicación en Heroku:
      6. heroku create
    6. Sube tu proyecto a Heroku:
      7. git push heroku master
    7. Abre tu aplicación desplegada:
      8. heroku open
